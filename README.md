# HydroCalc v1.2.7.12 🌿
**Calculadora Profesional de Nutrición Hidropónica**
HydroCalc es una herramienta avanzada diseñada para optimizar la formulación de soluciones nutritivas en sistemas hidropónicos y fertirriego. Esta aplicación permite a productores y técnicos calcular con precisión los aportes minerales necesarios, garantizando el equilibrio químico y maximizando el rendimiento del cultivo.
---
## 🚀 Guía de Instalación
Para instalar **HydroCalc** en su computadora, siga estos pasos:
1. **Descargar**: Localice el archivo `HydroCalc.zip` en la sección de descargas (o el archivo comprimido proporcionado).
2. **Extraer**: Descomprima el contenido del archivo ZIP en una carpeta local.
3. **Instalar**: 
   - Abra la carpeta extraída.
   - Haga doble clic en el archivo `setup.exe`.
   - Siga las instrucciones del asistente de instalación de **ClickOnce**.
4. **Ejecutar**: Una vez finalizada la instalación, podrá encontrar el acceso directo a **HydroCalc** en su menú de inicio.
